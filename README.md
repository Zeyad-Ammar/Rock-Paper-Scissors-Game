Rock Paper Scissors Game (C++)

A simple console-based Rock Paper Scissors game implemented in C++.
This project was built as a fun practice exercise while learning enums, structs, random number generation, and game logic in C++.

The player competes against the computer for several rounds, and the program determines the winner based on the classic Rock–Paper–Scissors rules.

 Features

Play Rock–Paper–Scissors against the computer
Choose the number of rounds (1–10)
Random computer moves
Display round results and choices
Track player wins, computer wins, and draws
Show the final game result
Visual console effects (colors and sound) depending on the winner
Option to play again
 Game Logic

The game uses the classic rules:

Paper beats Stone
Stone beats Scissors
Scissors beat Paper
Same choice → Draw

Each round:

Player selects a choice
Computer generates a random choice
The winner of the round is calculated
Scores are updated
Concepts Practiced

This project helped practice:

Enums
Structs
Random number generation
Game loops
Conditional logic
Console UI
State tracking across multiple rounds
Round results are displayed

At the end of the game, the program determines the final winner
