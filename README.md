<h1>Rock Paper Scissors Game (C++)</h1>

A simple console-based Rock Paper Scissors game implemented in C++.
This project was built as a fun practice exercise while learning enums, structs, random number generation, and game logic in C++.

The player competes against the computer for several rounds, and the program determines the winner based on the classic Rock–Paper–Scissors rules.

<h3>Game Logic</h3>

<h4>The game uses the classic rules</h4>

Paper beats Stone<br>
Stone beats Scissors<br>
Scissors beat Paper<br>
Same choice → Draw<br>

<h4>Each round</h4>

Player selects a choice<br>
Computer generates a random choice<br>
The winner of the round is calculated<br>
Scores are updated<br>
Concepts Practiced<br>

Structs<br>
Random number generation<br>

Conditional logic<br>

At the end of the game, the program determines the final winner
